## 使用建议

### 风格统一技巧

1. **固定参考游戏** - 始终引用相同的游戏（如《星露谷物语》）
2. **固定技术参数** - 始终指定"16位像素艺术"、"透明背景"
3. **固定尺寸** - 始终指定像素尺寸（如"64x64像素"）
4. **固定调色板** - 始终描述颜色范围（如"有限32色调色板"）

### 推荐 AI 工具

| 工具 | 适用场景 | 特点 |
|------|----------|------|
| Midjourney | 概念设计 | 风格统一性好 |
| Stable Diffusion | 批量生成 | 可训练LoRA保持风格 |
| DALL-E 3 | 快速原型 | 理解力强 |
| Leonardo.ai | 游戏素材 | 有游戏专用模型 |
| 文心一格 | 中文支持 | 国内可用 |
| 通义万相 | 中文支持 | 阿里出品 |

### 批量生成工作流

```
1. 使用基础提示词生成角色概念
2. 选择满意的风格
3. 固定该风格的关键描述词
4. 为不同角色/场景复用相同风格词
5. 在Sprite Processor中统一后处理
```

### 与框架集成流程

```
AI生成图片（使用中文提示词）
        ↓
auto-import/ai-generated/
        ↓
Sprite Processor（统一处理）
  - 去背
  - 统一尺寸
  - 切割精灵图
        ↓
Assets/Sprites/processed/
        ↓
Unity中使用
```

## 快速开始

### 生成角色

```bash
# 1. 使用提示词在AI工具中生成
# 2. 下载保存到 auto-import/characters/

# 3. 自动处理
./modules/cli/src/game-cli.sh sprite-process \
    --input auto-import/characters \
    --output Assets/Sprites/Characters
```

### 生成场景

```bash
# 1. 使用环境提示词生成
# 2. 保存到 auto-import/tilesets/

# 3. 自动处理（带切割）
./modules/cli/src/game-cli.sh sprite-process \
    --input auto-import/tilesets \
    --output Assets/Sprites/Tilesets \
    --slice --grid-size 32 32
```

---

**提示**：生成后务必使用框架的Sprite Processor统一处理，确保风格一致！
