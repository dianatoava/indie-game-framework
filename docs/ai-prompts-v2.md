## 分辨率计算公式

```
总宽度 = 列数 × 每格像素宽度
总高度 = 行数 × 每格像素高度
分辨率 = 总宽度 × 总高度
```

**常用配置：**

| 网格 | 每格大小 | 分辨率 | 适用场景 |
|------|---------|--------|---------|
| 6×4 | 256×256 | **1536×1024** | 22帧角色动画 |
| 4×4 | 256×256 | **1024×1024** | 16帧角色动画 |
| 8×8 | 128×128 | **1024×1024** | 64格瓦片地图 |
| 4×4 | 512×512 | **2048×2048** | 高清角色 |

## 快速计算工具

**填入参数，自动生成提示词末尾格式：**

```python
# 填入你的参数
cols = 6        # 列数
rows = 4        # 行数
cell_w = 256    # 每格宽度
cell_h = 256    # 每格高度

# 自动计算
total_w = cols * cell_w
total_h = rows * cell_h

print(f"分辨率：{total_w}×{total_h}")
print(f"严格遵循{cols}×{rows}网格布局，不要添加额外边框、网格线或文字")
```

## 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 不说格式 | 生成JPG，无透明背景 | 必须写 "格式：PNG" |
| 分辨率不对 | 切割时帧大小不一致 | 按公式计算正确分辨率 |
| 不说"严格遵循" | AI加边框或网格线 | 必须强调不要额外元素 |
| 不说"不要网格线" | AI自动加黑色分隔线 | 明确说不要网格线 |
